Markdown Guide

Learn how to format your posts and comments using Markdown

Text Formatting

Syntax Result Description
**bold** bold Make text bold
_italic_ or *italic* italic Make text italic
`code` code Inline code
~~strikethrough~~ strikethrough Line through text

Code Blocks

``` def factorial(n): if n == 0: return 1 return n * factorial(n-1) ``` 
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)

💡 Pro Tips
  • Use **bold** and _italic_ for emphasis
  • Use code blocks with backticks for showing code or mathematical formulas
  • Create clear structure with headings and sections
  • Use blockquotes to highlight important information
  • Preview your post before publishing to ensure formatting is correct
